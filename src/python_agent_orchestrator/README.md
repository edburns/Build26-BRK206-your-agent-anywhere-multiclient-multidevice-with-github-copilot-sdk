# py-skel

Reusable Python project skeleton for Windows + Dev Drive (E:).

## Quick start
- Create venv: `py -3.14 -m venv .venv`
- Activate (PowerShell): `. .\.venv\Scripts\Activate.ps1`
- Upgrade tooling: `python -m pip install --upgrade pip setuptools wheel`
- Install dev deps: `python -m pip install -e .[dev]`
